Sakai is an application template for Vue based on the [create-vue](https://github.com/vuejs/create-vue), the recommended way to start a Vite-powered Vue projects.

Visit the [documentation](https://sakai.primevue.org/documentation) to get started.

## Environment Setup

1. Copy `.env.example` to `.env`
2. Fill in your local values (see variable descriptions in `.env.example`)
3. Run the project normally

Only variables prefixed with `VITE_` are exposed to the frontend. Never place real secrets in `.env.example` or commit `.env` files.
